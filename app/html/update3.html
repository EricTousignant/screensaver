The Photo Screen Saver Extension Has Been Updated

 

There have been some significant changes in this version:

 

  • The API that is used to Access your Google Photos has changed

    • Your currently selected Albums are not compatible with the new API and have been removed. You will need to reselect  them.
    • You will be prompted to allow the Google Photo's API to access you Google Photos. You need to allow this. It says the extension can change your Google Photos, but that is not true - it is read only.
    • If you are having problems accessing your photos, exit Chrome from the menu. Do not just click on the 'X', and try again.
    • If you are still having problems, please uninstall and reinstall the extension.
    • There is now a limit to the number of API calls the extension can make per day. If this is exceeded, uncached photos will not be displayed and you will not be able to change your photo selections. The quota is reset at midnight Pacific time daily.
    • The extension will limit the total number of Albums you can select and the number of photos in an album. It you want to help out other users, you can limit the number of albums you use.
    • The refresh option of the Google Photos used to reload all the photos and they were reloaded every time you started Chrome and once a day. This is not the case anymore. If you add photos to an album you will need to deselect and select the Album in the extension to reload it.
    • The above 2 items are a bummer, but are intended to avoid exceeding the API quota. As I get some data on the total usage of the API, I will try to increase this. Please bear with me as I figure out the best policy.
    • Location information is no longer available from the API.
    • The album name will be displayed instead of your name.
    • You can now click on one of your photo's to display it in Google Photos.

  • Other changes

    • The code has been completely rewritten to support changes coming soon to Chrome. I have tested it, but there may be some issues. If so, please contact me from the "Help and Feedback" page.
    • Google has limited their GEO Location API. Location information of photos is currently not available. I will take a look at adding it back.
    • The time selection for the Keep Awake has been simplified.

 

Known issues:

 

  • When waking a sleeping screen by moving the mouse, the screensaver may display as black. Just move the mouse some more or press any key to close it.
